Sistema de aluno

- Solicitar o nome de um aluno
- Solicitar duas notas numéricas (tipo double) referente ao aluno
- Calcular a média aritmetica das duas notas

Exibir

- Nome do aluno
- A média obtida
- Status final
    - Aprovado se for maior ou igual a 6.0
    - Reprovado se a média for menor que 6.0
    - Para repositório
